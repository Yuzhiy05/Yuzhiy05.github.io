### vue入门

reactive 创建响应式对象

ref 创建响应式 基础类型 

html 标签

vue 标签 有这么一种写法
vue的标签 和html的标签类似 ,不过在vue中不叫标签叫组件 平时给html标签赋值时都是传静态字符串
vue这种加: 的写法其实就是动态绑定值  其实是v-bind:value 的简写写法
```html
<a-input :value="userinput">
```
